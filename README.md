# Data Redundancy Removal System

## 📌 Project Overview

The Data Redundancy Removal System is a cloud-based web application developed to identify and prevent duplicate data entries. The system validates user input and stores only unique records, improving database efficiency and accuracy.

This project was developed as part of the CodeAlpha Cloud Computing Internship.

---

## 🚀 Features

* Add and validate data entries
* Detect duplicate data instantly
* Prevent redundant storage
* Store only unique records
* Display stored unique data
* User-friendly responsive interface
